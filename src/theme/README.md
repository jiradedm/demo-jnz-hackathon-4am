# Winitch Brand Tokens

อิงจาก _docs/resource/Winitch Brand CI.txt และ _docs/resource/1.html

**Tailwind v4 — config ใน CSS อย่างเดียว** (ไม่มี tailwind.config.js)

- **สี / ฟอนต์ / Spacing / Animation**: กำหนดใน `src/index.css` ผ่าน `@theme { ... }`
- **Utility classes**: `.glass-card`, `.gradient-text`, `.grid-bg` ใน `src/index.css`
- **Dark mode**: ใช้ class บนต้นทาง (เช่น `<html class="dark">`) — กำหนดผ่าน `@custom-variant dark`
